# 🌾 Smart Farming Dashboard

A full-stack Smart Farming Dashboard built with **React, Node.js, Express, and Bootstrap**, featuring real-time sensor data visualization and interactive charts.  

This project simulates an IoT-based agriculture monitoring system inspired by modern precision farming solutions.

## 🧠 Features

- 📊 Real-time sensor data visualization
- 🌡️ Temperature, humidity, and soil moisture tracking
- 📈 Live updating charts using Recharts
- 🔄 Auto-refreshing backend API data
- 🎨 Responsive UI using Bootstrap
- ⚡ Full-stack architecture (frontend + backend)

---

## 🛠️ Tech Stack

### Frontend:
- React (TypeScript)
- Bootstrap
- Recharts

### Backend:
- Node.js
- Express.js
- REST API
